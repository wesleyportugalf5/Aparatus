## Requisitos Funcionais

- O botão de "Reservar" não deve fazer nada.
- O botão de "Copiar" telefone deve copiar o telefone para o clipboard.
- Os dados da barbearia devem ser recuperados do banco de dados, usando o ID recebido na URL.
- A imagem do banner da página no topo deve ser a imagem da barbearia no banco de dados (imageUrl).
- O botão de "Voltar" no banner da página deve voltar à tela anterior.
- Para cada serviço da barbearia recuperado do banco de dados, renderize o @components/service-item.tsx (crie o componente). Receba o serviço como prop.
